# 支持 / サポート

需要帮助或想反馈想法时，可以从以下入口开始：
困ったとき・提案したいときは、以下から始めてください。

---

## 1. 阅读文档 / ドキュメント

- 请先阅读 `README.md`（功能、架构、快捷入口、常用命令）。
  - まずは `README.md` をご確認ください。

---

## 2. 提交 Issue / Issue を作成

本仓库启用了 Issue 表单（Bug / Feature）。
本リポジトリでは Issue フォーム（Bug / Feature）を用意しています。

- Bug：选择 “Bug 报告 / バグ報告”
- Feature：选择 “功能建议 / 機能提案”

---

## 3. 安全问题 / セキュリティ

涉及漏洞、敏感信息、隐私问题，请遵循 `SECURITY.md`。
脆弱性・機微情報・プライバシーは `SECURITY.md` に従ってください。
