# 行为准则 / 行動規範

本项目致力于营造一个友好、包容、尊重的协作环境。
このプロジェクトは、友好的で包摂的、相互尊重の協力環境を目指します。

---

## 我们的承诺 / 私たちの約束

- 尊重每一位参与者，不因身份、经验或观点而贬低他人。
  - 参加者を尊重し、属性や経験、意見で貶めません。
- 以建设性方式交流，聚焦问题与解决方案。
  - 建設的に議論し、問題と解決策に集中します。

---

## 不被接受的行为 / 受け入れられない行為

- 人身攻击、侮辱、骚扰、歧视性言论。
  - 個人攻撃、侮辱、ハラスメント、差別的発言。
- 发布他人隐私信息或任何敏感信息。
  - 個人情報や機微情報の公開。
- 在沟通中故意制造对立、引战或刷屏。
  - 煽動、荒らし、スパム行為。

---

## 适用范围 / 適用範囲

适用于本仓库的 Issue、Pull Request、代码评审、讨论以及相关社区互动。
本リポジトリの Issue/PR/レビュー/議論など、関連する場面に適用されます。

---

## 举报与处理 / 報告と対応

如果你遇到不当行为：
不適切な行為を見かけた場合：

- 可通过 GitHub 的“Report content（报告内容）”功能进行举报（如可用）。
  - GitHub の「Report content」機能で通報してください（可能な場合）。
- 或在 Issue/PR 中 @ 维护者说明情况（请避免公开敏感信息）。
  - または Issue/PR でメンテナに @ してください（機微情報は公開しないでください）。
- 若涉及安全/隐私敏感内容，请参阅 `SECURITY.md` 进行非公开披露。
  - セキュリティ/プライバシーに関する場合は `SECURITY.md` に従って非公開で報告してください。

维护者将基于事实与影响采取合理措施，包括但不限于警告、删除内容、限制互动等。
メンテナは事実と影響に基づき、警告・投稿削除・制限等の対応を行うことがあります。
